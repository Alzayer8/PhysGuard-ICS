# Security policy

## Supported versions

Security fixes are provided for the latest public release.

| Version | Supported |
|---|---|
| 1.0.x | Yes |
| Earlier or private research editions | No |

## Reporting a vulnerability

Please use GitHub's **Report a vulnerability** private security-advisory flow for the
repository. Do not open a public issue with exploit details, sensitive telemetry, system
addresses, credentials, or operational information.

Include the affected version, impact, minimal reproduction, and suggested mitigation if
known. The maintainer will acknowledge a complete report as soon as practical, assess the
issue, and coordinate disclosure and a fix. Please allow reasonable remediation time
before public disclosure.

PhysGuard-ICS is offline analysis software. It must not be deployed as an autonomous
control, response, or safety system.
